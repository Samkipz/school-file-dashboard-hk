# Product Overview

## Purpose
School File Dashboard HK is a web-based school management portal providing staff and administrators with a centralized hub for school resources, announcements, media files, and communications.

## Key Features
- **Authentication** — Secure sign-in/sign-up via better-auth with session management
- **Dashboard** — Central landing page with activity feed, announcements, and upcoming events
- **Noticeboard** — School-wide notices and announcements
- **Staff Resources** — Internal documents and resources for staff
- **Media Files** — Centralized media asset management
- **General Section** — General school information and content
- **Sidebar Navigation** — Persistent nav across all authenticated pages

## Target Users
- School staff and administrators (HK school context)
- Users require authentication to access all dashboard features

## Use Cases
- Browsing and managing school announcements
- Accessing staff-only resources and documents
- Viewing and sharing media files
- Staying updated via activity feed and events calendar
